Getting Started

Welcome to Overture's how-to section. Here you'll find guides and tutorials to help you access and use the Overture datasets.

Learn and Build​

  • Access the data.
  • Query and explore the data.
  • Check out projects from the Overture community.
  • Visualize and build with the data.
  • Ask questions! Engage with Overture's schema working group and data working group on GitHub. Feedback from the community will help improve future iterations.

Dig Deeper​

  • Read a high-level overview of the schema for each data theme.
  • Consult the technical reference for schema details.
  • Learn about the simple feature model we use to describe Overture data.
  • Get to know the Global Reference Entity System (GERS).

Licensing​

Overture's data themes and schema are licensed under CC-BY 4.0. GERS IDs are licensed under CDLA Permissive v 2.0. Data that are associated with the themes through GERS can carry different licenses. However, if the data associated though GERS with an ODbL licensed theme constitutes a derivative dataset, it must be licensed under ODbL. See OpenStreetMap Community Guidelines for definitions of a derivative database.
