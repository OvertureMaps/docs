## Overview

**The `2025-07-23.0` release of Overture data and `v1.11.0` of the Overture schema are now available.** The datasets and release artifacts are available as GeoParquet files stored on both AWS and Azure. The paths are:

### Release data

**Microsoft Azure:**
```
az://overturemapswestus2.blob.core.windows.net/release/2025-07-23.0/
```

**Amazon S3:**
```
s3://overturemaps-us-west-2/release/2025-07-23.0/
```

## Theme-specific updates

:::info
The base, buildings, divisions, places, and transportation themes are in GA. The addresses theme is in alpha. 
:::

### Addresses

- no updates


### Base

- added OSM tags for `quay`, `railway`, `artwork`

### Buildings

- added a new data source to populate favelas in Brazil: DadosAbertos
- added `is_land` geometries to the `division_area` type, `country` subtype


### Divisions

- added new "land" polygons (`is_land=TRUE`) for regions that have access to oceans

### Places

- no updates

### Transportation

- fixed a small bug in the route `source_id` formatting
- added new data from TomTom in Turkey
