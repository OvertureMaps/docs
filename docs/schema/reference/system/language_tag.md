# LanguageTag (`string`)

A BCP-47 language tag.

As described in `Tags for Identifying Languages, BCP-47`_.

.. _Tags for Identifying Languages, BCP-47: https://www.rfc-editor.org/rfc/bcp/bcp47.txt

For example:

- `"en"`
- `"en-US"`
- `"fr-CA"`
- `"zh-Hant-TW"`

## Constraints

- `LanguageTagConstraint`
