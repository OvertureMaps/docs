# RailFlag

Simple flags that can be on or off for a railway segment.

Specifies physical characteristics and can overlap.

## Values

- `is_bridge`
- `is_tunnel`
- `is_under_construction`
- `is_abandoned`
- `is_covered`
- `is_passenger`
- `is_freight`
- `is_disused`
