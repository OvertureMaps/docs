# BuildingSubtype

Broadest classification of the type and purpose of a building.

This broad classification can be refined by `BuildingClass`.

## Values

- `agricultural`
- `civic`
- `commercial`
- `education`
- `entertainment`
- `industrial`
- `medical`
- `military`
- `outbuilding`
- `religious`
- `residential`
- `service`
- `transportation`
