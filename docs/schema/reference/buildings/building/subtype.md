# Subtype

A broad category of the building type/purpose.

When the current use of the building does not match the built purpose, the subtype
should be set to represent the current use of the building.

## Values

- `agricultural`
- `civic`
- `commercial`
- `education`
- `entertainment`
- `industrial`
- `medical`
- `military`
- `outbuilding`
- `religious`
- `residential`
- `service`
- `transportation`
