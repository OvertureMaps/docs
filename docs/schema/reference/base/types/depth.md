# Depth

Depth below surface level of the feature in meters.

Underlying type: `int32`

## Constraints

- `≥ 0`
- `≥ -2147483648` (from [`int32`](../../system/primitive/primitives.md))
- `≤ 2147483647` (from [`int32`](../../system/primitive/primitives.md))
