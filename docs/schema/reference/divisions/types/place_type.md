# PlaceType

Category of the division from a finite, hierarchical, ordered list of categories (e.g., country,
region, locality, etc.) similar to a Who's on First placetype.

## Values

- `country`
- `dependency`
- `macroregion`
- `region`
- `macrocounty`
- `county`
- `localadmin`
- `locality`
- `borough`
- `macrohood`
- `neighborhood`
- `microhood`
