## Subtypes

儘管區劃主題的架構包含了全面的子類型列表，但目前數據中僅有部分子類型可用。

目前，在 `division_area` 特徵類型中可用的子類型有：

- `country`
- `dependency`
- `region`
- `county`
- `localadmin`
- `locality`
- `neighborhood`

一個國家是獨立主權的最大單位。國家的邊界（如果已知）由 [divsion boundary](/schema/reference/divisions/division_boundary) 特徵表示。例如：美國

dependency 是一個不是一個國家的子區域的地方，但依賴於母國進行國防、護照控制等。例子：波多黎各

  </TabItem>

region 是大多數國家內最大的行政單位，並且是第一級行政區。例子：阿爾伯塔

county 是一種第二級行政區。例如：紐約的金斯縣
  </TabItem>

  <TabItem value="localadmin" label="localadmin" default>
    <table>

Localadmin 是世界某些地區的政府層級，包含沒有自己管理權限的地方或有人居住的地點。這種層級常見於歐洲，但並不僅限於此。地方行政區是第三級行政區。例子：[巴黎](https://spelunker.whosonfirst.org)
  </TabItem>

  <TabItem value="locality" label="locality" default>

Locality是指有人居住的地點，可能擁有或不擁有自己的行政權限。（如果它屬於地方行政區，那麼就沒有自己的管理權限。）地方可能是第三級行政區。例子：台北
  </TabItem>

Neighborhood 是指位於城市、城鎮、郊區或鄉村地區的地理化社區，有時可能僅由一條街道及其沿街的建築物組成。例子：Cobble Hill
