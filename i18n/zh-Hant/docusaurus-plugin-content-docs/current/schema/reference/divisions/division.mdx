# division

division 是一個從特定政治角度來看被承認的官方或非官方的組織。

## 子類型

雖然 divisions 主題的 schema 包含了一個全面的子類型列表，但目前只有一部分子類型可用於資料中。

目前，在 `division` 特徵類型中提供了以下子類型：

- `country`
- `dependency`
- `region`
- `county`
- `localadmin`
- `locality`
- `neighborhood`
- `microhood`
