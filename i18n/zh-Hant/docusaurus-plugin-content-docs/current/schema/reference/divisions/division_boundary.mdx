# Division Boundary

Division boundary 是 [division](/schema/reference/divisions/division) 之間的邊界。

Division boundary 是通常由官方定義的兩個 [division](/schema/reference/divisions/division) 之間的邊界。考慮到地緣政治問題，一些邊界代表獨立的爭議線或條約線，因此不一定與 division 的邊界重合。

## 子類型

雖然 divisions 主題的 schema 包含了全面的子類型列表，但目前數據中僅提供了部分子類型。

目前，`division_boundary` 特徵類型中可用的子類型如下：

- `country`
- `region`
- `county`
