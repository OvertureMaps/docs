# Connector 

Connector 在 [Segment](/schema/reference/transportation/segment) 之間創建物理連接。

Connector 通過定義兩個或多個[ Segment ](/schema/reference/transportation/segment)之間的物理連接點，幫助確定交通網絡的拓撲結構。

除了它們的點幾何和所有 Overture 特性所需的核心屬性外，Connector 沒有其他屬性。
