#  address 

Overture  address 類型是由當地授權來源通常發佈的公開 address 資料集的彙編。在我們的初始版本中，我們使用了由 [OpenAddresses](https://openaddresses.io) 收集和分發的資料集。

 address 的初始架構相當簡單，預期將擴展到更複雜的 address 方案。

##  Features 類型
 address 主題只有一種 Features 類型。
- `address` 是一個點 Features 類型，表示單個 address 點。
