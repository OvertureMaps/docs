## 跨主題的複雜屬性

一些跨主題使用的 Features 屬性是複雜的，需要進一步解釋。

- **[名稱](/schema/concepts/names)**：在適用的情況下，複雜屬性如名稱在所有主題中始終使用相同的模式
- **[範疇和規則基屬性](/schema/concepts/scoping-rules)**：範疇屬性縮小其父屬性的範圍，僅適用於父屬性的特定實例；規則基屬性是一個規則數組，規則是包含至少一個範疇屬性的對象
- **[製圖](/schema/concepts/cartography)**：製圖屬性是一組可選的「提示」，用於顯示 Overture 資料在地圖上的方式
