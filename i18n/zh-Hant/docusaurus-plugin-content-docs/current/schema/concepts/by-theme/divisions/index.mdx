## 概覽
Overture 分區主題包括代表現實世界中人類聚落的特徵，如國家、地區、州、城市和城鎮。

## 特徵類型
分區主題有三個特徵類型。
- `division` 是一個特徵類型，代表一個官方或非官方的人類組織——國家、地區、省份、城市、社區等——從某個政治角度來看。它具有點幾何，顯示與該特徵最常關聯的位置的近似位置
- `division_area` 是一個特徵類型，捕捉分區特徵所屬的土地區域或陸地和領海（海洋）的形狀。它具有多邊形或多重多邊形幾何
- `division_boundary` 是一個特徵類型，代表兩個分區特徵之間的共享邊界。它具有線串或多重線串幾何。邊界的幾何要麼完全是非海洋的，要麼完全是海洋的。海洋邊界是非海洋邊界向水域延伸
