## 概要
Overture 地址主題是由地方授權來源通常發布的開放地址資料集的匯編。在我們的初始版本中，我們使用由 [OpenAddresses](https://openaddresses.io) 收集和分發的資料集。

初步的地址模式相當簡單，預計將擴展到更複雜的地址方案中。

##  Features 類型
地址主題有一種類型的 Features 。
- `address` 是一種點 Features 類型，代表單個地址點。
