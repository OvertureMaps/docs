## 願景

[Overture Maps Foundation 的成員公司](https://overturemaps.org/about/members/)正在合作建設數據集和架構。他們共享一個共同的願景：

**解決核心問題，促進邊緣發展。** Overture 架構並不能解決所有問題。它為最基本的「核心」使用案例提供完整的即時解決方案。同時，該架構的可擴展結構使得各種其他使用案例（「邊緣」）得以實現。

**填補空隙創新。** 地圖社群已經可以使用許多優秀的工具、標準和做法。Overture 架構重用這些現有的解決方案，以最大化兼容性，並專注於解決尚未解決的痛點。

**向前兼容即向後兼容。** 沒有任何設計能夠預見未來，但好的設計能夠通過添加功能而不破壞現有功能來保持相關性。Overture 架構可以以向後兼容的方式增強。

**永遠開源。** Overture 架構和數據格式旨在與免費和開源工具兼容，避免依賴專有技術。
