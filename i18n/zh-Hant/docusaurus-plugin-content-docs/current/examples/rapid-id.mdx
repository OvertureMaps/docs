在此範例中，我們將 Overture Maps 地點資料載入 Rapid，以便可視化和檢查幾何形狀及其屬性。

要打開 Rapid v2，請前往 [https://rapideditor.org/edit](https://rapideditor.org/edit)。

使用 + 或 - 快捷鍵、滑鼠滾輪等功能，縮放到您想查看 Overture 地點資料的區域。

點擊側邊欄中的 '地圖圖資' 按鈕或按 'F' 快捷鍵。然後，點擊 '自訂地圖資料' 旁邊的三個點圖示。

![Rapid 中的地圖資料 面板](/img/rapid/visualizing-places.png)

在自訂地圖資料設定中。將以下 URL 複製到輸入框中：

 <QueryBuilder query="https://overturemaps-tiles-us-west-2-beta.s3.amazonaws.com/__PMTILES_OVERTURE_RELEASE/places.pmtiles" language="text"></QueryBuilder>

然後點擊 'OK'。

等待一下子後會看到地圖上開始顯示一些地點資料。這些地點是可點擊和可檢查的：

![Rapid 顯示地點資料 疊加](/img/rapid/rapid-places-data.png)

另外，如果 URL 格式如下，Rapid 可以自動載入資料 。

 <QueryBuilder query="https://rapideditor.org/edit#data=https://overturemaps-tiles-us-west-2-beta.s3.amazonaws.com/__PMTILES_OVERTURE_RELEASE/places.pmtiles" language="text"></QueryBuilder>
