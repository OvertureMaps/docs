[DuckDB](https://duckdb.org/) 是一款分析工具，允許您查詢遠端文件並僅下載您所需的數據。

## 安裝

[在本地安裝 DuckDB](https://duckdb.org/docs/installation/)。您將需要擴展功能來處理雲端的空間數據。使用 DuckDB CLI，請執行以下操作：

1. `INSTALL SPATIAL;` 來安裝 [duckdb_spatial](https://github.com/duckdblabs/duckdb_spatial) 擴展。
2. `INSTALL httpfs;` 或 `INSTALL azure;` 來從 Amazon S3 ([httpfs](https://duckdb.org/docs/guides/import/s3_import.html)) 或 Microsoft Azure Blob Storage ([azure](https://duckdb.org/docs/extensions/azure)) 讀取數據。

## 示例查詢

### 山脈
此查詢從 Overture 地點數據集中選擇山脈類別的 POI，並將其輸出到 GeoJSON 文件中。


<QueryBuilder query={ConfidentMountains}></QueryBuilder>

:::tip
To write the data to a shapefile, replace the last two lines with:

```sql
    ) TO 'overture_places_mountains_gt90.shp'
WITH (FORMAT GDAL, DRIVER 'ESRI Shapefile');
```
:::

### 在底特律的建築

此查詢從 Overture 建築數據集中提取底特律地區的建築幾何形狀和選定屬性。

此查詢從 divisions 數據集中抓取賓夕法尼亞州的縣級幾何形狀，並將其輸出到 GeoJSON 文件中。
