您需要創建一個帳戶才能使用 [Synapse](https://azure.microsoft.com/en-us/products/synapse-analytics/) 和其他 Microsoft Azure 服務。

在您的 Azure 帳戶中，創建一個 [Synapse 工作區](https://learn.microsoft.com/en-us/azure/synapse-analytics/get-started-create-workspace)。然後，您可以執行此範例查詢來得到西雅圖的地點：
