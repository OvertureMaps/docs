---
title: 特徵型態的穩定性 #Stability by Feature Type
---
Overture 的所有特徵都有一個唯一的 ID，對於某些特徵類型，這些 ID 被稱為 Overture ID，並註冊到全球實體參考系統 (GERS)。註冊到 GERS 的 Overture ID 用來在多個 Overture 發布版本中保持穩定使其在合理的錯誤容忍範圍內。當無法保證穩定性時，我們將在 [GERS 更新日誌](../changelog) 中提供後續追蹤。

| 主題 | 類型 | Stable Overture ID |
|-------|------|-----------|
| addresses | address | No |
| base | infrastructure | No |
| base | land | No |
| base | land_cover | No |
| base | land_use | No |
| base | water | No |
| buildings | building | Yes |
| buildings | building_part | No |
| divisions | division | Yes |
| divisions | division_area | Yes |
| divisions | boundary | No |
| places | place | Yes |
| transportation | segment | Yes |
| transportation | connector | Yes |
